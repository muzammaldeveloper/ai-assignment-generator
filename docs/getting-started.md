# Getting Started

## Prerequisites

- Python 3.11+
- Node.js 20+
- Redis (for async worker)

## Backend Quick Start

1. Go to Backend
2. Create and activate virtual environment
3. Install dependencies
4. Copy .env.example to .env
5. Run Flask app
6. Optionally run Celery worker

## Frontend Quick Start

1. Go to Frontend
2. Install dependencies
3. Run development server

## Verify Setup

- Backend health endpoint returns success
- Frontend loads login and dashboard pages
